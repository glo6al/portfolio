Bug Bash

Role: Frontend Developer and UX/UI Designer

This project was collaborative classwork using front-end and back-end technologies to create a community for new developers to share and fix their coding bugs. The tech stack used for this project includes Bootstrap, Sequelize, Dotenv and Bcrypt, Node.js, and Express.js. For my part, I designed the front-end of the app, creating the mock-ups, wireframes, and user flows. I also coded the projects HTML, CSS and contributed to the front-end JavaScript. Users can create an account and log in to view, comment, and upvote posts made by other developers.

Furbaby Finder

Role: UX/UI Designer

Furbaby Finder is collaborative classwork and a web app that connects pets in need of adoption with potential adopters. For my part, I designed the front-end of the app, which involved creating mock-ups, wireframes, user stories, and other documentation. We built the app using jQuery, APIs, and AJAX. Users can search for adoptable pets by location and filter their search results by animal type, gender, and radius.

Peak Microgym

Role: Developer

Peak Functional Fitness is a deployed project using the no-code platform Weebly. It is a multi-page website with a homepage, an about page, online store, calendar, and a contact page. The website is integrated with Printful.com and PushPress.com to support e-commerce capabilities. Branding and colors were my brain children, and are applied throughout the website. I wrote all the content, using a light-hearted tone to match the branding.

GORX Fitness

Role: Product Manager/Stakeholder

The GORX Fitness website showcases my work as a product manager and the low-code development work of Ivo Valentjin. It was designed using Webflow, a powerful and user-friendly low-code platform.
